## Misc Notes / Comments - 

- git log --oneline
- git log --pretty=format:"%h - %an, %ar : %s" (ar indicates relative author time of commit)
- git log --pretty=format:"%h - %an, %ai : %s" (ai indicates exact author time of commit)
